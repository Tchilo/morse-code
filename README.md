## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites
```
Ruby
```

### Setup
```
git clone git@github.com:Tchilo/morse-code.git
cd morse-code.git
```

### Run Program
```
ruby decode_morse_code.rb
```
